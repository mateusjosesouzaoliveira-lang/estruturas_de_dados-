Este repositório reúne a resolução de exercícios desenvolvidos durante cursos da plataforma Alura, com foco no aprendizado prático da linguagem Python.

Os exercícios abordam conceitos fundamentais e progressivos, permitindo consolidar a base lógica e estrutural da programação.

🧠 Conteúdos praticados
📋 Estruturas de dados (listas, tuplas e dicionários)
🔁 Estruturas de repetição e controle
🧮 Manipulação de números e operações básicas
📂 Leitura e escrita de arquivos
📊 Manipulação de arquivos CSV
⚡ Compreensões (list/dict comprehension)
